# 常用 Markdown 語法

1. 標題：
```
# 一級標題
## 二級標題
### 三級標題
```
2. 列表：
```
  - 項目 1
  - 項目 2
    - 子項目 A
    - 子項目 B
```
    ![upload successful](\blog.github.io\images\pasted-2.png)
3. 連結：
```
[連結文字](http://www.example.com)
```
4. 圖片：
```
![圖片描述](http://www.example.com/image.jpg)
```
5. 自動連結：
```
<http://www.example.com>
```
6. 粗體和斜體：
```
**粗體文字**
*斜體文字*
```

7. 引用：
```
> 引用文本
```
8. 分隔線：
```
	---
```
9. 表格：
```
| 標題1 | 標題2 |
|-------|-------|
| 內容1 | 內容2 |
```
10. 跳脫字符：
```    \```
11. HTML 元素：
使用 ```<br> ```換行。

12. 刪除線：
```
~~刪除這段文字~~
```
13. 改字體顏色:
```
<font color="red">這是紅色文字</font>
```
